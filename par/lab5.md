### 4. Контрольные вопросы

#### 4.1. Виды мониторов

1. **ЭЛТ (CRT)** — электронно-лучевая трубка (устаревшие)
2. **ЖК (LCD)** — жидкокристаллические
3. **Светодиодные (LED)** — подвид ЖК с LED-подсветкой
4. **Плазменные (PDP)** — газоразрядные панели
5. **OLED** — органические светодиоды
6. **QLED** — квантовые точки (разновидность LED)
7. **Сенсорные** — с поддержкой касаний

#### 4.2. Конструкция ЖК-мониторов

ЖК-монитор состоит из следующих основных элементов:

- **Жидкокристаллическая матрица** — панель с пикселями из жидких кристаллов
- **Подсветка** — LED-лампы или люминесцентные лампы (CCFL)
- **Поляризационные фильтры** — два фильтра для управления светом
- **Цветовые фильтры** — RGB-фильтры для каждого субпикселя
- **Стеклянные подложки** — между которыми находятся жидкие кристаллы
- **Управляющие транзисторы** (TFT) — для каждого пикселя
- **Контроллер** — управляет отображением изображения
- **Корпус** — защитный пластиковый корпус

Принцип работы: жидкие кристаллы под действием напряжения меняют свою ориентацию, пропуская или блокируя свет от подсветки через поляризационные и цветные фильтры.
